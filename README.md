## 🚀 Sobre el proyecto

**Sports Hub Games** es una plataforma web que transforma el entretenimiento en una experiencia interactiva:

✨ Juega  
🧠 Aprende  
🏆 Compite  

Diseñada para ofrecer **diversión rápida con beneficios cognitivos**.
